# 🌍 AI Language Translator

An AI-powered language translation web application built using Streamlit.

## Features

- 🌍 Translate text into multiple languages
- 🔍 Auto Language Detection
- 🔊 Text-to-Speech
- 📥 Download Translation
- 🕒 Translation History
- 🎨 Light/Dark Theme

## Technologies

- Python
- Streamlit
- Deep Translator
- gTTS
- LangDetect

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
